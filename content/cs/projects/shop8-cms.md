# Shop8 CMS

Shop 8 je česká firma specializující se na vývoj e-commerce řešení na míru pro středně velké firmy.
::

::div{.row}

## O projektu

Shop 8 se zaměřuje na vytváření flexibilních a škálovatelných e-commerce řešení pro středně velké firmy, které již překonaly limity standardních šablonových platforem.
::

::div{.row}

## Klíčové funkce

- **Vývoj CMS na míru** - Přizpůsobené řešení pro každého klienta
- **E-commerce řešení** - Kompletní online obchodní platformy
- **Škálovatelná architektura** - Růst spolu s byznysem klienta
- **Zaměření na růst byznysu** - Řešení pro rozvíjející se firmy

::
